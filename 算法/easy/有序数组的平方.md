## 题目描述
```
给定一个按非递减顺序排序的整数数组 A，返回每个数字的平方组成的新数组，要求也按非递减顺序排序。

 

示例 1：

输入：[-4,-1,0,3,10]
输出：[0,1,9,16,100]
示例 2：

输入：[-7,-3,2,3,11]
输出：[4,9,9,49,121]
 

提示：

1 <= A.length <= 10000
-10000 <= A[i] <= 10000
A 已按非递减顺序排序。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/squares-of-a-sorted-array

```

## 思路

>   循环A数组计算每个元素的平方，将平方的数组排序返回。程序如下：
>
>   ```java
>class Solution {
>       public int[] sortedSquares(int[] A) {
>        for (int i = 0; i < A.length; i++) {
>               A[i] = A[i] * A[i];
>           }
>            Arrays.sort(A);
>            return A;
>        }
>    }
>    ```
>
